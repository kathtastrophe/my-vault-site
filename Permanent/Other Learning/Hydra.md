# Hydra

![[Hydra.png|200]]


**Hydra** is a monster in Greek mythology. When Hydra is decapitated, two more heads grow in place. 

Hydra was defeated by Hercules. Hercules sealed the monsters necks by using fire, preventing regeneration.
